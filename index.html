Mickey.js

Mouses are smarter now

Why Mickey?

  • No dependecy
  • jQuery-like API
  • Event-based
  • Easy to use

Try it yourself:

Text

Cool! I have a follower! 
            Mickey('.text').text( 'The Follower >_<', { color : '#DD4814', fontSize: '14px', textShadow: '0 0 1px #DD4814' } );
So texts have a heart, too 
            Mickey('.effect').text( 'Heartbeat!', { color : 'red', textShadow: '0 0 1px red' }, 0, 0, { name : 'heartbeat'} );

Relative Mouse Position

Positions! Positions! 
  		      var relative = document.getElementById('relative');
  		      relative.addEventListener('mousemove', function(){
  		        
  		        var pos = Mickey('#relative').relative();
  		        relative.innerHTML= 'top:' + pos.top;
  		        relative.innerHTML+= '  left:' + pos.left;
  		        
  		      })
